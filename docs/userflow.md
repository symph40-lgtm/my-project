## 플로우 1 - 아침 방어 브리핑 "장 시작 전에 오늘은 뭘 조심해야 하지?"

**목표 순간:** 출근 전 또는 장 시작 전, 사용자가 "오늘은 무리해도 되는 날인지, 방어해야 하는 날인지"를 판단한다.  
**상태 변화:** 막연한 불안 -> 오늘의 시장 위험과 내 행동 기준을 아는 상태

**단계 시퀀스**

| 단계 | 화면 | 기능 | 주요 행동 | 보조 |
|:--:|------|------|-----------|------|
| 1 | 오늘의 방어 브리핑 | 전날 미국장 요약 | 앱을 열고 오늘의 영향도 확인 | 로그인 없이 첫 요약 확인 |
| 2 | 근거 상세 | 매크로 해석 | 왜 위험/중립/호재인지 근거 펼치기 | 금리, 환율, 유가, 주요 지표 |
| 3 | 내 포트폴리오 입력 | 포트폴리오 반영 | 보유 종목과 현금 비중 입력 또는 수정 | 처음에는 간단 입력 |
| 4 | 민감도 진단 | 한국장 영향도 판정 | 내 포트폴리오에 민감한 리스크 확인 | 섹터, 종목, 레버리지 노출 |
| 5 | 행동 가이드 | 가격 방어 코칭 | 오늘의 매수/보유/방어 기준 저장 | 과매수 방지 문장 제공 |

**화면**

### 오늘의 방어 브리핑

```text
┌────────────────────────────┐
│ 오늘 한국장 영향도          │
│ [위험]                      │
│                            │
│ 금리 상승 + 반도체 약세 +   │
│ 환율 부담이 겹쳤습니다.     │
│                            │
│ 지금 볼 것 3가지            │
│ 1. 반도체 선물 흐름          │
│ 2. 원/달러 환율              │
│ 3. 외국인 수급               │
│                            │
│ [근거 보기] [내 포트 점검]   │
└────────────────────────────┘
```

- **역할:** 사용자가 가장 먼저 봐야 할 결론을 짧게 제시한다.
- **보이는 정보·요소:** 영향도, 핵심 근거 3개, 근거 보기, 포트폴리오 점검.
- **주요 행동:** `[내 포트 점검]` 선택.

### 내 포트폴리오 입력

```text
┌────────────────────────────┐
│ 내 포트폴리오               │
│ 보유 종목: [삼성전자, SK하이닉스] │
│ 현금 비중: [ 20% ]          │
│ 레버리지: [없음 v]          │
│                            │
│ [민감도 보기]               │
└────────────────────────────┘
```

- **역할:** 개인화된 판단에 필요한 최소 정보를 받는다.
- **보이는 정보·요소:** 보유 종목, 현금 비중, 레버리지 여부.
- **주요 행동:** `[민감도 보기]` 선택.

### 행동 가이드

```text
┌────────────────────────────┐
│ 오늘의 행동 가이드           │
│                            │
│ 신규 매수: 관망 또는 축소     │
│ 보유: 급락 첫날 매도 금지     │
│ 레버리지: 추가 진입 금지      │
│                            │
│ 방어 문장                   │
│ "오늘은 수익을 먹는 날보다    │
│ 손실을 키우지 않는 날입니다." │
│                            │
│ [오늘 기준 저장]             │
└────────────────────────────┘
```

- **역할:** 시장 판단을 사용자의 실제 행동 기준으로 바꾼다.
- **보이는 정보·요소:** 매수, 보유, 레버리지 기준과 방어 문장.
- **주요 행동:** `[오늘 기준 저장]` 선택.

**핵심 가치:** 많은 데이터를 찾아보지 않아도, 장 시작 전에 오늘의 위험과 행동 기준을 빠르게 세운다.

## 플로우 2 - 장중 판단 보정 "지금 흔들리는데 팔아야 하나?"

**목표 순간:** 장중 급락, 급반등, 이슈 업데이트가 있을 때 사용자가 충동 매매 전에 판단을 다시 점검한다.  
**상태 변화:** 흔들림과 공포 -> 바뀐 정보와 유지할 기준을 구분하는 상태

**단계 시퀀스**

| 단계 | 화면 | 기능 | 주요 행동 | 보조 |
|:--:|------|------|-----------|------|
| 1 | 장중 변화 알림 | 이슈 진행 추적 | 중요한 변화 알림 확인 | 소음 알림은 제외 |
| 2 | 오늘의 방어 브리핑 | 시간대별 판단 | 아침 판단이 유지되는지 확인 | 오전/장중/마감 전 상태 |
| 3 | 근거 상세 | 변동성 패턴 감지 | 급락이 단기 소음인지 추세 위험인지 확인 | 2~3일 연속 하락 패턴 |
| 4 | 민감도 진단 | 포트폴리오 반영 | 내 보유 종목의 위험 노출 확인 | 섹터별 영향 |
| 5 | 행동 가이드 | 가격 방어 코칭 | 팔기, 버티기, 일부 축소 중 선택 | 결정 전 방어 문장 |

**화면**

### 장중 변화 알림

```text
┌────────────────────────────┐
│ 판단 변경 가능성 있음        │
│                            │
│ 오전보다 환율 부담이 커졌고, │
│ 반도체 수급이 약해졌습니다.  │
│                            │
│ 기존 판단: 중립             │
│ 현재 판단: 위험 주의         │
│                            │
│ [무엇이 바뀌었나 보기]       │
└────────────────────────────┘
```

- **역할:** 사용자의 판단을 바꿀 만큼 중요한 변화만 호출한다.
- **보이는 정보·요소:** 기존 판단, 현재 판단, 바뀐 이유.
- **주요 행동:** `[무엇이 바뀌었나 보기]` 선택.

### 근거 상세

```text
┌────────────────────────────┐
│ 바뀐 근거                   │
│                            │
│ 1. 환율: 부담 확대           │
│ 2. 반도체: 외국인 매도 강화  │
│ 3. 미국 선물: 회복 실패      │
│                            │
│ 단기 소음보다 위험 쪽에      │
│ 무게가 실립니다.             │
│                            │
│ [내 보유 영향 보기]          │
└────────────────────────────┘
```

- **역할:** 사용자가 뉴스 제목이 아니라 판단 근거를 보게 한다.
- **보이는 정보·요소:** 변한 지표, 해석, 다음 행동 연결.
- **주요 행동:** `[내 보유 영향 보기]` 선택.

### 행동 가이드

```text
┌────────────────────────────┐
│ 지금 할 일                  │
│                            │
│ 신규 매수: 멈춤              │
│ 보유: 손절 기준 전까지 유지  │
│ 일부 축소: 레버리지 있으면 검토 │
│                            │
│ 결정 전 질문                │
│ "지금 파는 이유가 데이터인가, │
│ 공포인가요?"                │
│                            │
│ [오늘 기준 유지] [일부 축소]  │
└────────────────────────────┘
```

- **역할:** 급한 감정 반응을 판단 절차로 늦춘다.
- **보이는 정보·요소:** 지금 할 일, 결정 전 질문, 선택 버튼.
- **주요 행동:** `[오늘 기준 유지]` 또는 `[일부 축소]` 선택.

**핵심 가치:** 장중 변동성 속에서도 사용자가 "바뀐 정보"와 "내 감정"을 구분하고, 무리한 추격 매수나 공포 매도를 줄인다.

## 플로우 3 - 마감 전 리스크 정리 "내일은 어떻게 준비하지?"

**목표 순간:** 장 마감 전, 사용자가 다음 거래일의 리스크를 예상하고 보유, 현금, 레버리지 기준을 정한다.  
**상태 변화:** 하루 결과만 본 상태 -> 내일의 위험과 대비 행동을 정한 상태

**단계 시퀀스**

| 단계 | 화면 | 기능 | 주요 행동 | 보조 |
|:--:|------|------|-----------|------|
| 1 | 마감 전 점검 | 시간대별 판단 | 오늘 시장 결과와 내일 리스크 확인 | 마감 전 요약 |
| 2 | 민감도 진단 | 포트폴리오 반영 | 보유 종목의 내일 민감도 확인 | 종목/섹터별 노출 |
| 3 | 근거 상세 | 매크로 해석 | 다음 날 영향을 줄 이벤트 확인 | CPI, 고용, 금리, 환율 |
| 4 | 행동 가이드 | 가격 방어 코칭 | 현금, 보유, 레버리지 기준 결정 | 결정 저장 |
| 5 | 오늘의 방어 브리핑 | 전날 미국장 요약 | 다음 날 아침 확인 항목 예약 | 알림 설정 |

**화면**

### 마감 전 점검

```text
┌────────────────────────────┐
│ 마감 전 점검                │
│                            │
│ 오늘 하락은 아직 하루짜리입니다. │
│ 다만 환율과 금리 부담은      │
│ 내일까지 이어질 수 있습니다. │
│                            │
│ 내일 확인할 것              │
│ 1. 미국 고용 지표            │
│ 2. 반도체 선물               │
│ 3. 환율 1차 저항선           │
│                            │
│ [내일 기준 정하기]           │
└────────────────────────────┘
```

- **역할:** 오늘 결과와 내일 리스크를 한 장으로 묶는다.
- **보이는 정보·요소:** 오늘 요약, 이어질 수 있는 리스크, 내일 확인 항목.
- **주요 행동:** `[내일 기준 정하기]` 선택.

### 행동 가이드

```text
┌────────────────────────────┐
│ 내일 기준                   │
│                            │
│ 현금: 20% 이상 유지          │
│ 신규 매수: 지표 확인 후       │
│ 보유: 반등 시 추격 매수 금지  │
│ 레버리지: 사용하지 않음       │
│                            │
│ [기준 저장] [아침 알림 켜기]  │
└────────────────────────────┘
```

- **역할:** 다음 날 흔들리지 않도록 미리 행동 기준을 고정한다.
- **보이는 정보·요소:** 현금, 신규 매수, 보유, 레버리지 기준.
- **주요 행동:** `[기준 저장]` 선택.

### 아침 알림 설정

```text
┌────────────────────────────┐
│ 내일 아침 확인              │
│                            │
│ 오전 8:40에                 │
│ "오늘 한국장 영향도"를       │
│ 다시 알려드릴게요.           │
│                            │
│ [알림 켜기]                 │
└────────────────────────────┘
```

- **역할:** 마감 전 판단을 다음 날 아침 루틴으로 연결한다.
- **보이는 정보·요소:** 알림 시간, 알림 내용.
- **주요 행동:** `[알림 켜기]` 선택.

**핵심 가치:** 사용자가 하루 손익에 끌려가지 않고, 다음 거래일에 지켜야 할 기준을 미리 정한다.

## v1에서 기본값으로 처리할 것

- 로그인과 계좌 연결은 v1 필수 흐름에서 제외하고, 직접 입력 또는 관심 종목 입력으로 시작한다.
- 매수/매도 직접 주문 기능은 제공하지 않는다.
- 예측 수익률보다 `위험/중립/호재`, `근거`, `행동 제한`을 우선한다.
- 고수의 전략 설명은 핵심 화면에 짧은 보조 문장으로만 넣고, 별도 콘텐츠 화면은 P1로 둔다.

## PRD로 넘길 결정 포인트

- 첫 화면의 1순위 정보는 `오늘 한국장 영향도`로 둔다.
- 포트폴리오 입력은 종목, 현금 비중, 레버리지 여부까지만 v1에 포함한다.
- 알림은 모든 뉴스가 아니라 판단 변경 가능성이 있는 변화에만 보낸다.
- 매수/매도 권고는 직접 명령보다 행동 가이드와 방어 문장 중심으로 표현한다.
- 서비스의 핵심 차별점은 "시장 데이터를 더 많이 보여주는 것"이 아니라 "오늘의 방어 행동을 정하게 해주는 것"이다.
