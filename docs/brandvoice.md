## 확정 정보

- **서비스 이름:** 마켓가드
- **한 줄 무드:** 시장이 흔들릴 때 사용자의 감정보다 먼저 원칙을 세워주는 차분하고 엄격한 방어 코치.
- **이름 메모:** "마켓가드"는 시장을 예측해 맞히는 서비스보다, 위험 신호를 먼저 막아주고 사용자의 충동적 행동을 가드한다는 포지션을 선명하게 전달한다.
- **가벼운 이름 확인:** 일반 검색, Play 스토어, App Store 검색어 기준으로 동일 맥락의 뚜렷한 충돌은 확인되지 않았다. 정식 상표 검토는 별도 필요하다.

## 1. Voice Summary

Sounds like a disciplined market coach who translates complex signals into calm, actionable guardrails.

마켓가드는 친절하게 달래기보다, 사용자가 피하고 싶은 숫자를 보게 만들고 다음 행동의 경계를 정해준다. 말투는 차분하지만 흐릿하지 않고, 경고는 단호하지만 공포를 키우지 않는다.

## 2. Core Personality Traits

**차분한**
- We are: 급락장에서도 호흡을 낮추고 확인할 순서를 제시한다.
- We are not: 불안을 과장하거나 과도하게 위로하지 않는다.
- This sounds like: "먼저 시장 위험부터 확인하세요. 오늘은 신규 매수보다 비중 점검이 우선입니다."
- This does NOT sound like: "큰일 났어요. 지금 당장 뭔가 해야 합니다."

**엄격한**
- We are: 원칙을 어기는 행동에는 분명히 제동을 건다.
- We are not: 사용자가 듣고 싶은 낙관만 말하지 않는다.
- This sounds like: "반등만 보고 추격 매수하기에는 근거가 부족합니다."
- This does NOT sound like: "느낌이 좋다면 조금 더 담아도 괜찮을 수 있어요."

**근거 중심의**
- We are: 금리, 환율, 유가, 수급, 섹터 흐름을 행동 판단과 연결한다.
- We are not: 점수나 예측만 던지고 설명을 생략하지 않는다.
- This sounds like: "환율과 금리가 함께 올라 위험 신호가 겹쳤습니다."
- This does NOT sound like: "오늘 시장 점수는 62점입니다."

**보호적인**
- We are: 손실 만회, 공포 회피, 레버리지 과다 사용을 줄이는 방향으로 말한다.
- We are not: 매매를 자극하거나 성공 확신을 팔지 않는다.
- This sounds like: "지금은 수익을 더 키우는 구간보다 손실 확대를 막는 구간입니다."
- This does NOT sound like: "오늘 놓치면 다시 오지 않을 기회입니다."

**실전적인**
- We are: 사용자가 출근 전, 점심시간, 장마감 전에 바로 판단할 수 있게 압축한다.
- We are not: 전문 용어를 길게 늘어놓거나 리포트처럼 말하지 않는다.
- This sounds like: "장초반에는 관망, 장마감 전에는 레버리지 비중을 다시 보세요."
- This does NOT sound like: "복합 매크로 환경의 다변량 상관성이 확대되는 국면입니다."

## 4. Vocabulary Guide

### USE

- **위험도:** 오늘의 판단을 가장 먼저 압축할 때 사용한다.
- **근거:** 결론 뒤에 반드시 붙이는 설명 단위다.
- **행동 제한선:** 충동 매매를 막는 핵심 표현으로 사용한다.
- **비중 점검:** 매수/매도보다 먼저 확인해야 할 행동으로 사용한다.
- **레버리지 축소:** 위험 구간에서 직접적인 방어 행동으로 사용한다.
- **현금화:** 감정적 도피가 아니라 리스크 관리의 선택지로 말한다.
- **관망:** 아무것도 하지 않는 것이 아니라 원칙 있는 대기 상태로 말한다.
- **확인 필요:** 알림과 경고의 기본 표현으로 사용한다.
- **추격 매수:** 피해야 할 행동을 명확히 지칭한다.
- **손실 확대 방어:** 서비스의 핵심 효용을 설명할 때 사용한다.
- **시장 영향도:** 미국장과 매크로를 한국장 판단으로 번역할 때 사용한다.
- **오늘의 방침:** 사용자가 하루 동안 지킬 결론으로 사용한다.

### AVOID

- **대박, 급등 확정, 무조건:** 수익 보장처럼 들리므로 금지한다.
- **지금 안 사면 늦습니다:** 조급함을 자극하므로 금지한다.
- **완벽한 예측:** 서비스의 포지션과 맞지 않는다.
- **추천 종목:** 판단 코치보다 리딩방처럼 보일 수 있다.
- **꿀팁:** 시장 위험을 가볍게 보이게 한다.
- **멘탈 관리만 하세요:** 데이터 기반 방어라는 차별점이 흐려진다.
- **괜찮을 수도 있어요:** 위험 구간에서 모호한 안심을 준다.
- **AI가 알아서 판단:** 사용자 책임과 근거 확인을 흐린다.
- **공포에 사라:** 원칙보다 구호처럼 들린다.
- **손절하세요:** 맥락 없이 직접 지시하는 표현은 피한다.
- **초고수, 비밀 전략:** 신뢰보다 과장된 마케팅처럼 들린다.
- **승률:** 예측 서비스로 오해될 수 있다.

## 5. Tone Adaptation Rules

| 맥락 | 톤 조절 | 예시 문구 |
|---|---|---|
| 온보딩 / 첫 화면 | 단호함보다 신뢰와 구조를 먼저 준다. | "마켓가드는 오늘의 시장 위험도와 행동 제한선을 먼저 정리합니다." |
| 빈 상태 | 입력을 강요하지 않고, 입력하면 좋아지는 판단을 설명한다. | "보유 종목을 추가하면 오늘 특히 민감한 섹터를 함께 확인할 수 있습니다." |
| 성공 / 완료 피드백 | 과한 축하보다 완료 사실과 다음 확인 지점을 알려준다. | "포트폴리오를 반영했습니다. 오늘 위험도 판단에 보유 비중을 함께 보겠습니다." |
| 오류 / 안내 | 사용자를 탓하지 않고 다음 행동을 짧게 제시한다. | "데이터를 불러오지 못했습니다. 잠시 후 다시 확인해 주세요." |
| 즉시 알림 | 짧고 선명하게, 공포를 키우지 않는다. | "위험 신호가 겹쳤습니다. 신규 매수 전 비중을 먼저 확인하세요." |
| 위험 브리핑 | 엄격하게 말하되 이유와 행동을 함께 제시한다. | "오늘은 위험 구간입니다. 환율과 금리가 동시에 부담으로 작용해 레버리지 축소를 우선 검토하세요." |
| 우호 브리핑 | 낙관을 과장하지 않고 조건을 함께 말한다. | "우호 신호가 우세하지만, 급등 종목 추격 매수는 제한하세요." |
| 장마감 전 | 다음날 리스크까지 포함해 더 보수적으로 말한다. | "장마감 전에는 내일 갭하락 가능성을 함께 봐야 합니다. 현금 비중을 너무 낮추지 마세요." |

## 6. Style Rules

- 존댓말을 기본으로 쓴다. "~하세요", "~이 우선입니다"처럼 명확한 문장을 사용한다.
- 문장은 짧게 쓴다. 한 문장에 하나의 판단만 담는다.
- 결론은 `위험 / 중립 / 우호` 중 하나로 먼저 말하고, 뒤에 근거와 행동을 붙인다.
- 전문 용어는 쓰되, 사용자가 바로 행동으로 옮길 수 있는 말로 번역한다.
- 이모지는 사용하지 않는다. 알림과 경고의 무게를 유지한다.
- 느낌표는 쓰지 않는다. 긴급함은 단어가 아니라 정보 구조로 전달한다.
- 매수/매도 표현은 직접 지시보다 `검토`, `제한`, `축소`, `유지 가능`을 우선한다.
- 숫자가 있으면 가능한 한 구체적으로 쓴다. 단, 근거 없는 정밀한 확률은 만들지 않는다.

## 7. Do's and Don'ts

**Do:** 먼저 오늘의 시장 위험도를 말한다.  
**Don't:** 뉴스와 지표를 길게 나열한 뒤 결론을 숨기지 않는다.

**Do:** 경고에는 항상 이유와 다음 행동을 붙인다.  
**Don't:** "위험합니다"만 말하고 사용자를 불안 속에 두지 않는다.

**Do:** 사용자의 충동을 막는 문장을 선명하게 쓴다.  
**Don't:** 더 사도 되는 이유를 억지로 찾아주지 않는다.

**Do:** 위험 구간에서는 보수적으로 말한다.  
**Don't:** 반등 가능성을 수익 기회처럼 과장하지 않는다.

**Do:** 포트폴리오 맥락을 반영해 개인화된 경고를 만든다.  
**Don't:** 모든 사용자에게 같은 시장 코멘트만 보여주지 않는다.

**Do:** 시장을 맞히는 서비스가 아니라 판단을 방어하는 서비스로 말한다.  
**Don't:** 예측 정확도나 승률을 브랜드의 중심으로 삼지 않는다.

**Do:** 친절하지만 단호하게 말한다.  
**Don't:** 지나치게 다정해서 원칙이 흐려지게 만들지 않는다.

**Do:** "오늘 하지 말아야 할 일"을 명확히 제시한다.  
**Don't:** 사용자가 알아서 해석해야 하는 추상 조언으로 끝내지 않는다.
